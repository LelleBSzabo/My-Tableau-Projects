# 📊 Tableau Retail & Sales Data Visualisation Project

This project was completed during a **Data Technician Bootcamp** and focuses on analysing and visualising datasets using **Tableau**. The goal of the project was to transform raw data into meaningful insights through **interactive dashboards, filters, calculated fields, and visual storytelling**.

Using Tableau’s visual analytics tools, the project explores patterns in **retail, sales, and demographic data**, enabling users to quickly identify trends, compare performance, and make data-driven decisions.

---

## 🚀 Project Overview

In this project, Tableau was used to:

- Build **interactive dashboards** to explore datasets
- Visualise sales and demographic trends using multiple chart types
- Apply **filters and slicers** to allow dynamic user interaction
- Create **calculated fields** to generate new insights from existing data
- Communicate findings through **data storytelling and visual analytics**

The project demonstrates how Tableau can be used to turn complex datasets into **clear, interactive business insights**.

---

## 🛠️ Tableau Skills Demonstrated

### Interactive Dashboards
- Designed dashboards combining multiple visualisations
- Enabled user interaction through **filters and controls**
- Structured layouts to support **clear storytelling and insights**

### Data Exploration Tools
- **Filters** to dynamically explore different data segments
- **Slicers and dropdown selectors** for user-driven analysis
- Cross-filtering between visualisations within dashboards

### Calculated Fields
Used calculated fields to derive additional insights from the data, such as:

- Aggregated metrics
- Conditional logic for data categorisation
- Percentage calculations for distributions

---

## 📊 Visualisations Created

A variety of chart types were used to communicate different aspects of the data:

- **Bar Charts** – Compare values across categories
- **Line Charts** – Display trends over time
- **Pie Charts** – Show proportional distributions
- **Scatter Plots** – Explore relationships between variables
- **Maps** – Visualise geographic patterns
- **Custom Visualisations** – Additional charts to explore insights

These charts were combined into **interactive dashboards** to provide a comprehensive view of the data.

---

## 📈 Dashboard Features

Key dashboard functionality included:

- Interactive **country and category filters**
- Dynamic updates across multiple charts
- Comparative visualisations for easier analysis
- Clear titles and layout for storytelling

Users can interact with the dashboard to explore:

- Regional trends
- Category performance
- Changes over time
- Distribution of demographic or categorical variables

---

## 🔍 Key Insights

Through visual analysis of the datasets, several trends and insights were identified, such as:

- Clear **regional differences in performance and outcomes**
- Observable **trends over time** within key metrics
- Relationships between variables revealed through scatter plots
- Distribution differences across categories and groups

These insights demonstrate how visual analytics can help organisations **quickly identify patterns and support decision-making**.
